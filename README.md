# NR GOLD

A sleek and interactive website for NR GOLD, a gold jewelry business. The website showcases their jewelry collections, provides company information, and allows customers to easily get in touch.

## Note

The full source code for this project is not publicly shared due to proprietary reasons and company confidentiality. This repository contains the **production build** of the website for public viewing.

## Live Demo

[View Website](https://nrgold.com)

## Features

- **Showcases jewelry collections:** Browse different products with images and descriptions  
- **About the company:** Learn NR GOLD's story and mission  
- **Contact information:** Easily access phone, email, and location details  
- **Responsive design:** Works well on desktops, tablets, and mobile devices  
- **User-friendly interface:** Clean navigation and layout for a smooth browsing experience  

## Tech Stack

- React  
- HTML  
- CSS  
- JavaScript  

## How It Works

- The website is a static React app with multiple pages for jewelry collections, company info, and contact details  
- Users can navigate seamlessly between pages using the navigation menu  
- Layout and styling are responsive, ensuring the website looks good on all screen sizes
